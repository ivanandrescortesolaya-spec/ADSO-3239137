Modelos de Datos

MER (Modelo Entidad-Relación) y MR (Modelo Relacional)

MER - Modelo Entidad-Relación

Representa entidades, atributos y relaciones de forma conceptual usando rectángulos, elipses y rombos.

MR - Modelo Relacional

Representa tablas con columnas, tipos de datos, claves primarias (PK) y foráneas (FK).

Despliegue con Docker

Contenedorización de bases de datos

¿Qué es Docker?

Plataforma de contenedorización que permite empaquetar aplicaciones y sus dependencias en contenedores ligeros y portables.

Ventajas para BD

  • Entornos consistentes y reproducibles
  • Fácil despliegue y escalabilidad
  • Aislamiento de recursos
  • Versionado de configuraciones

Ejemplo docker-compose.yml

version: '3.8'
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: secret
      MYSQL_DATABASE: sena_db
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql

  mysql_data:
